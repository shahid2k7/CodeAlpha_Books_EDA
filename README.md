# CodeAlpha_Books_EDA 📊
## Task 2: Exploratory Data Analysis (EDA)

### 📋 Project Description
As part of my Data Analytics internship at **CodeAlpha**, I performed a comprehensive Exploratory Data Analysis (EDA) on a dataset of 500 books. This project focuses on cleaning raw web-scraped data and extracting meaningful insights using Python.

### 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Environment:** Jupyter Notebook / VS Code

### 🔍 Key Tasks & Findings
1. **Data Cleaning:** - Removed currency symbols (Â£) using Regex.
   - Converted categorical ratings (One, Two, Three...) into numerical formats for analysis.
2. **Missing Value Analysis:** Identified and handled any null entries in the dataset.
3. **Statistical Summary:** Calculated central tendencies (Mean, Median) for book prices.
4. **Visualizations:**
   - **Price Distribution:** Visualized the price spread using Histograms.
   - **Rating Correlation:** Used Boxen plots to see if higher-rated books are more expensive.
   - **Heatmaps:** Checked the correlation between numeric variables.

### 🚀 Insights
- The average price of books in the collection is around £35.
- There is no direct strong correlation between the rating and the price of the book.
- Identified potential outliers in the premium pricing segment.
